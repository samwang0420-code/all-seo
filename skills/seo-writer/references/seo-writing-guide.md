# SEO写作参考

## 搜索意图分类（决定文章写法）

### 1. 信息型（Informational）
用户想知道某件事是什么、怎么工作。
**写法重点：** 解释清楚，有具体例子，允许一定篇幅。
**开头：** "什么是XX？" 直接切入。
**结尾：** "如果你想进一步了解，可以…" 或下一篇文章的引导。

### 2. 导航型（Navigational）
用户想找某个具体网站/品牌。
**写法重点：** 不是SEO文章的主战场，适合品牌词保护。
**注意：** 除非用户搜索的就是你的品牌词，否则不要硬凹导航型。

### 3. 商业调查型（Commercial Investigation）
用户在做购买决策前在研究对比。
**写法重点：** 对比要具体（功能/价格/体验），不能只说"很好"。
**必须包含：** 至少一个竞品对比或真实用户评价引用。
**结尾：** "如果你想了解具体价格/购买方式，可以…" 自然引向转化。

### 4. 交易型（Transactional）
用户已经准备好行动了（购买/注册/下载）。
**写法重点：** 行动路径清晰，CTA明确，不要绕弯子。
**开头：** 直接说"想买XX，看这篇就够了"。
**结尾：** 具体的下一步（链接/表单/联系方式）。

## 关键词研究基础

### 主关键词选择标准

1. 月搜索量 ≥ 500（太低没流量，太高竞争太大）
2. 商业价值匹配（搜索这个词的人，跟你的目标客户重合）
3. 你有实际内容可以支撑（不能硬凑内容匹配）

### 长尾关键词

- 定义：3个词以上、更具体的搜索词
- 例子："2000元以内投影仪推荐"（长尾）vs "投影仪"（主词）
- 价值：搜索量小但转化率高、竞争低、意图明确
- 策略：主词打排名，长尾词打转化

### 关键词布局原则

```
标题：主关键词放最前面
第一段：100字内出现主关键词
正文：自然出现每段一次，不要堆砌
结尾：可以再提一次主关键词
URL：包含主关键词（英文或拼音）
```

## 文章结构参考

### 标准五段式（最通用）

```
第一段（100字内）：痛点/场景切入 + 主关键词
第二段（200-300字）：XX是什么 + 核心特点概览
第三段（400-600字）：具体讲解/对比/使用场景 + 长尾关键词
第四段（300-500字）：实操建议/步骤/注意事项
第五段（100-200字）：总结 + CTA
```

### 清单式（适合"XX推荐"类）

```
开头：为什么需要XX（痛点）
主体：X个选择，每个包含：名称 + 核心特点 + 适合人群 + 缺点
结尾：我的建议 + CTA
```

### 教程式（适合"How to"类）

```
开头：这个事我帮你们测了一遍，发现…
第一步：（先做什么）
第二步：（再做什么）
…
常见坑：XXX
我的建议：YYY
CTA
```

## EEAT 写作技巧

### Experience（经验）

不要只说"这个产品好"，要说你**怎么知道它好的**：

- ❌ "这款空气净化器效果很好"
- ✅ "我在卧室放了三天，用专业仪器测了PM2.5，从40降到了8"

- ❌ "这家公司服务不错"
- ✅ "我跟他们客服聊了三次，每次响应都在10分钟以内"

### Expertise（专业）

数据要具体，来源要可查：

- ❌ "数据显示，80%的用户对这个产品满意"
- ✅ "他们官网公开的2024年用户评价里，827条评论平均分4.6"

引用权威来源要具体：

- ❌ "根据行业报告"
- ✅ "IDC发布的2024年中国SaaS市场报告中"

### Authoritative（权威）

承认竞品的优点，不要把自己写成唯一正确答案：

- ❌ "如果你不想买最贵的，这款是唯一的选择"
- ✅ "如果你预算充足，XX这个型号在XX方面更强；如果你预算有限，这款是我们测试下来性价比最高的"

### Trustworthy（可信）

不夸大，局限性要主动说：

- ❌ "这款产品没有任何缺点"
- ✅ "这款产品有两个明显的不足：…但对于XX需求的人来说，这两个问题影响不大"

## 常见SEO文章类型写法

### ① XX推荐类（最常见）

关键点：
- 覆盖产品/品牌要全，不能只推自己家的
- 每个推荐要有具体理由，不能只说"好"
- 要有价格区间（用户关心）
- 要有适合人群（帮助用户对号入座）

### ② XX对比类（高转化）

关键点：
- 对比维度要具体（不能只对比价格）
- 要有明确的推荐结论（不能"各有优劣"敷衍了事）
- 缺点要提，越提越显得客观

### ③ How to / 教程类

关键点：
- 每个步骤要具体，不能跳步
- 要有"常见错误/坑"的提醒（读者很吃这个）
- 要有"需要准备什么"的清单

### ④ 解答类（问答型）

关键点：
- 第一个问题要高频且有搜索量
- 回答要直接，不能绕弯子
- 可以扩展1-2个相关问题，增加内容深度
