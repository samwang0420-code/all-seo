Frequently Asked Questions

What does GE Appliances E5 error mean?
The GE Appliances E5 error indicates temperature sensor error on your refrigerator. This code is triggered when the appliance's control system detects this specific fault during its self-check cycle.
Can I fix the E5 error myself?
Yes - most homeowners can resolve this error with basic steps in about 30-60 min.
How do I reset the E5 error?
Unplug the appliance for 60 seconds, then plug it back in. This resets the control board and may clear the error. If it returns immediately, there is an underlying fault that needs to be addressed.
How much does repair cost?
DIY parts typically cost $10-$50. Professional service runs $100-$300 depending on your location.
